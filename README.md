# mana-seeds-rust-tool

A small Rust desktop tool (built with `eframe`/`egui`) for loading an image and overlaying a configurable grid.

## Features

- Open common image formats (`png`, `jpg`, `jpeg`, `bmp`, `gif`)
- Configure grid rows, columns, cell size, and offset
- Apply/regenerate grid overlay
- Zoom in/out on the canvas
- Drag to pan the canvas

## Run

```bash
cargo run
```

## Build checks

```bash
cargo check
```
